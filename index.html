Help / User Manual

Welcome to PTSC/THRI Crew Dashboard. This guide will help you navigate:

  • Login: Enter your username and password.
  • Tabs: Use the sidebar to switch between Crew Joining, Arrivals, Updates, Memo, Training, P&I, Chatboard.
  • Add/Edit/Delete: Click "Add New" to insert data, "Edit" to modify, "Delete" to remove.
  • PDF Reports: Click "PDF" button in table rows for single item, or "Generate Monthly PDF" for whole dashboard.
  • Chatboard: Type message and click Send.
  • Sticky Notes: Use the bottom text area to write private notes.
