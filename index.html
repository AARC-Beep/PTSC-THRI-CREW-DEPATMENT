PTSC/THRI Dashboard - User Manual

1. Login

Enter your Username and Password to access the dashboard. Only authorized users can login.

2. Dashboard Overview

Quick summary cards show recent activity for Crew Joining, Arrivals, Updates, Memo, Training, and P&I Events. Click the card to view details in each tab.

3. Adding Data

Click the Add New button in each tab (Crew Joining, Arrivals, etc.), fill out the form, then click Save.

4. Editing Data

Click the Edit button next to an entry. Update fields and click Save to apply changes.

5. Deleting Data

Click the Delete button next to an entry. Data will be moved to the archive sheet.

6. Generating PDF
  • Individual Record: Click PDF next to the entry to download it.
  • Monthly Report: Click Generate Monthly PDF in Dashboard to download a report with all entries.
7. Chatboard

Type your message in the input box and click Send. Chat updates live for all logged-in users.

8. Sticky Note

Use the sticky note at the bottom for personal notes. Notes are private to your session.

9. User Roles

Some tabs like Training and P&I are admin-only. Non-admin users cannot access them.

10. External Links

The sidebar contains external links for quick access to FlightAware, MSN News, Zoom Earth, MARINA, DMW, and OWWA.

Need Help?

Contact the system administrator for access issues or additional support.
